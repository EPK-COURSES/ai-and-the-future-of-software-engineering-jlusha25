# Project Executive Summary — Topic 8

This document provides a high-level comparative overview of how generative AI technologies and automated systems have been integrated into the software engineering workflows of Microsoft and Google. 

### Core Matrix Comparison

| Feature | GitHub Copilot (Microsoft) | AI Code Review / Critique (Google) |
| :--- | :--- | :--- |
| **Primary Use Case** | Real-time inline code generation & boilerplate automation | Automated pull request summaries & style guide compliance |
| **Measured Benefit** | Up to 55% speed increase in standard developer tasks | Significant reduction in human code reviewer fatigue |
| **Key Limitation** | Risk of generating security vulnerabilities & hallucinations | Requires strict internal corporate network isolation |
| **Human Role** | Crucial for validation, testing, and structural architecture | Final approval authority on all automated code suggestions |
